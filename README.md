## colocar no git
###	`git init`
###	`git status`
###	`git add . `
###	`git commit -m "initial commit"`
usando git bash no windows:
###	`ls -a ~/.ssh`
###	`eval "$(ssh-agent -s)"`
###	`ssh-add ~/.ssh/id_rsa`
voltar pro promt de comando:
###	`clip.exe <~/.ssh/id_rsa.pub`
e colar em ssh key no github
###	`ssh -T git@github.com`
###	`git remote add origin <ssh>`
